特性：
一个Label标签控件。支持背景色、边框颜色、边框宽度、文字、文字间距、标签内边距等属性的设置；
支持静态设置和代码中动态修改。
效果：
当设置icon显示未false的时候，会有单纯文字的初始效果。
引入：推荐通过jitpack
注意事项：
在使用是的时候，需要参考mainActivity中的调用方式，不然可能没有效果
自定义属性：

文本

labelTextColor
labelTextSize

labelIconVisible
labelIconSize

labelBackgroundEndColor
